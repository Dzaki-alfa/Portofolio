Robot Line Follower

Kategori: Robotics
Tahun: 2026
Durasi: 1 bulan
Tingkat: Menengah

Komponen Utama

Arduino Nano Sensor IR (5) Motor DC Driver L298N Baterai LiPo

Spesifikasi

  • Dimensi 15cm x 12cm x 8cm
  • Berat 350 gram
  • Kecepatan 0.5 m/s
  • Waktu Operasi 45 menit

Deskripsi Proyek

Robot Line Follower adalah robot otonom yang dirancang untuk mengikuti garis hitam di atas permukaan putih. Sistem menggunakan sensor inframerah untuk mendeteksi garis dan mikrokontroler Arduino Nano sebagai pengendali utama.

Latar Belakang

Proyek ini dikembangkan untuk mempelajari sistem kontrol PID dan pemrograman embedded system. Robot dirancang agar stabil saat berjalan lurus maupun saat menghadapi tikungan tajam.

Tantangan

Tantangan utama dalam proyek ini adalah melakukan tuning parameter PID agar robot tidak mengalami overshoot saat melewati tikungan serta menjaga kestabilan navigasi.

Fitur Utama

Kecepatan Variabel

PID mengatur kecepatan motor secara real-time.

Navigasi Presisi

Mampu mengikuti tikungan dengan akurasi tinggi.

Arduino Nano

Mikrokontroler efisien dan mudah diprogram.

Low Power

Konsumsi daya optimal untuk penggunaan lama.

Source Code 



int ENB = 9; // Speed motor kiri
int majuB = 6; // Arah maju motor kiri
int mundurB = 7; // Arah mundur motor kiri

int ENA = 3; // Speed  motor kanan
int majuA = 4; //Arah maju motor kanan
int mundurA = 5; // Arah mundur motor kanan

int pb = 2;

int sensor1 = A0; // Sensor kiri luar
int sensor2 = A1; // Snensor kiri dalam
int sensor3 = A2; // Sensor tengah
int sensor4 = A3; // Sensor kanan dalam
int sensor5 = A4; // Sensor kanan luar

#define Sensor1 digitalRead(sensor1) // Makro
#define Sensor2 digitalRead(sensor2) // Makro
#define Sensor3 digitalRead(sensor3) // Makro
#define Sensor4 digitalRead(sensor4) // Makro
#define Sensor5 digitalRead(sensor5) // Makro

bool sistemNyala = false;
bool tombolTerakhir = LOW;

void setup() {
  pinMode(sensor1, INPUT); // Deklarasi  sensor mode INPUT
  pinMode(sensor2, INPUT); // Deklarasi  sensor mode INPUT
  pinMode(sensor3, INPUT); // Deklarasi  sensor mode INPUT
  pinMode(sensor4, INPUT); // Deklarasi  sensor mode INPUT
  pinMode(sensor5, INPUT); // Deklarasi  sensor mode INPUT

  pinMode(ENA, OUTPUT); // Deklarasi  kecepatan motorA mode OUTPUT
  pinMode(mundurA, OUTPUT); // Deklarasi arah motorA mode OUTPUT
  pinMode(majuA, OUTPUT); // Deklarasi arah motorA mode OUTPUT

  pinMode(ENB, OUTPUT); // Deklarasi  kecepatan motorB mode OUTPUT
  pinMode(mundurB, OUTPUT);  // Deklarasi arah motorA mode OUTPUT
  pinMode(majuB, OUTPUT);  // Deklarasi arah motorA mode OUTPUT

  pinMode(pb, INPUT);
  
  analogWrite(ENB, 150); // Kecepatan maju motorB
  analogWrite(ENA, 150); // Kecepatan maju motorA
}

void loop() {

  bool tombol = digitalRead(pb);

  // Deteksi tekan tombol (LOW → HIGH)
  if (tombolTerakhir == LOW && tombol == HIGH) {
    sistemNyala = !sistemNyala; // toggle ON / OFF
    delay(200); // debounce sederhana
  }

  tombolTerakhir = tombol;

  // JIKA SISTEM MATI
  if (!sistemNyala) {
    analogWrite(ENA, 0);
    analogWrite(ENB, 0);
    return;
  }

  digitalWrite(majuB, HIGH); // Setup arah maju motorB
  digitalWrite(majuA, HIGH); // Setup arah maju motorA
  digitalWrite(mundurB, LOW);
  digitalWrite(mundurA, LOW);

//  Masuk logika IF per sensor
  if(Sensor3 == HIGH) {
    analogWrite(ENB, 150);
    analogWrite(ENA, 150);
  }else if(Sensor1 == HIGH){
    analogWrite(ENB, 50);
    analogWrite(ENA, 150);
  }else if(Sensor2 == HIGH){
    analogWrite(ENB, 90);
    analogWrite(ENA, 150);
  }else if(Sensor4 == HIGH){
    analogWrite(ENB, 150);
    analogWrite(ENA, 90);
  }else if(Sensor5 == HIGH){
    analogWrite(ENB, 150);
    analogWrite(ENA, 50);
  }

  // Logika jika sensor LOW semua, motor akan mundur
  if (Sensor1 == LOW && Sensor2 == LOW && Sensor3 == LOW && Sensor4 == LOW && Sensor5 == LOW) {
    analogWrite(ENB, 0);
    analogWrite(ENA, 0);
  }
  
}
