Smart Office IoT System

Kategori: IoT Automation
Tahun: 2025
Tingkat: Menengah

Komponen Utama

ESP32 Sensor PIR Sensor Suhu DHT Relay Module Buzzer Lampu Kipas

Fitur Sistem

  • Monitoring Suhu ruangan
  • Kontrol Lampu otomatis
  • Kontrol Kipas berdasarkan suhu
  • Notifikasi Buzzer alarm

Deskripsi Sistem

Smart Office adalah sistem otomatisasi ruangan berbasis Internet of Things (IoT) yang dirancang untuk meningkatkan efisiensi energi serta kenyamanan dalam ruang kerja. Sistem ini memanfaatkan berbagai sensor untuk memonitor kondisi lingkungan dan mengontrol perangkat elektronik secara otomatis.

Dengan menggunakan mikrokontroler ESP32, sistem mampu membaca data dari sensor seperti sensor gerakan dan sensor suhu, kemudian memproses data tersebut untuk mengaktifkan perangkat seperti lampu, kipas, maupun buzzer.

Cara Kerja Sistem

Sensor PIR digunakan untuk mendeteksi keberadaan manusia di dalam ruangan. Ketika sensor mendeteksi gerakan, sistem akan mengaktifkan lampu secara otomatis sehingga ruangan menjadi terang.

Sensor suhu akan membaca temperatur ruangan secara berkala. Jika suhu melebihi batas tertentu, sistem akan menyalakan kipas untuk menjaga kenyamanan ruangan.

Buzzer digunakan sebagai indikator sistem aktif atau sebagai notifikasi ketika terjadi kondisi tertentu pada sistem.

Alur Sistem

    Sensor membaca kondisi lingkungan

    Data dikirim ke mikrokontroler ESP32

    ESP32 memproses logika sistem

    Output diaktifkan (lampu, kipas, buzzer)

Tujuan Proyek

Proyek ini dibuat untuk mempelajari implementasi sistem IoT dalam otomatisasi ruangan. Selain itu, sistem ini juga bertujuan untuk meningkatkan efisiensi penggunaan energi listrik dengan mengontrol perangkat secara otomatis berdasarkan kondisi lingkungan.
